About

Hello, my name is Alex Machin. I’m an aspiring front-end web developer, who is currently studying at Rotherham College.

My journey into web development started just over a year ago, since then I’ve enjoyed every moment of it. Throughout the first year I spent majority of my time completing various courses on udemy and watching various YouTubers.

During this time I've picked up some of the best web development skills, some of the them are displayed below.

GULP JS

Gulp is a JavaScript based automation tool which starts my server, makes my files web ready, and reruns tasks when specified. I don’t know why I didn’t invest in gulp earlier.

SASS

SASS enables me to optimise my stylesheet for perfection by allowing modules, mixins and variables to be used. Styling is now scalable and maintainable for all my current and future project.

Node.js

Node.js allows me to access the npm registry to access thousands of plugins developed to improve the functionality of a workflow or a project. Node is offically my favourite web development tool.
